1주차: AI 기초 및 데이터 분석

문제:

AI를 위해 사용되는 데이터는 크게 세 가지 종류로 나눌 수 있습니다. 이 세 가지 데이터 종류를 모두 나열하고, 각각의 예시를 한 가지씩 제시하시오.

모범 답안:

  • 정형 데이터 (Structured Data): 미리 정해놓은 형식과 구조에 따라 저장된 데이터입니다.
    예시: 관계형 DB, 스프레드시트, CSV 데이터 등
  • 반정형 데이터 (Semi-structured Data): 구조에 따라 저장된 데이터이지만 데이터의 형식과 구조가 변경될 수 있는 데이터입니다.
    예시: HTML, XML, JSON 등
  • 비정형 데이터 (Unstructured Data): 정해진 구조 없이 저장된 데이터입니다.
    예시: Text, Image, Sound, Multimedia 데이터 등

문제:

머신러닝의 주요 학습 유형 세 가지를 제시하고, 각각의 학습 유형이 어떤 데이터를 기반으로 학습하는지 또는 학습의 목적을 간략하게 설명하시오.

모범 답안:

  • 지도 학습 (Supervised Learning): 정답이 있는 데이터를 학습하는 유형입니다.
  • 비지도 학습 (Unsupervised Learning): 정답이 없는 데이터를 학습하며, 비슷한 특징의 데이터끼리 군집화(클러스터링)하거나 차원 축소 등을 목적으로 합니다.
  • 강화 학습 (Reinforcement Learning): 환경과의 상호작용을 통해 스스로 학습하며, 최적의 행동 정책을 찾는 것을 목적으로 합니다.

문제:

머신러닝 모델에서 발생하는 문제 중 하나인 과대적합(Overfitting)은 무엇을 의미하며, 이를 해결하거나 완화하기 위한 방법으로 정규화(Regulation)를 사용합니다. kingcounty-house-sales 실습에서 다룬 두가지 정규화 방법 및 각각의 특징을 설명하시오.

모범 답안:

과대적합(Overfitting)은 모델이 훈련 데이터를 너무 과도하게 학습하여 훈련 데이터에 대해서는 매우 높은 성능을 보이지만, 실제 새로운 데이터나 보지 못한 데이터에 대해서는 성능이 떨어지는 현상을 의미합니다.

정규화(Regulation)는 모델의 복잡도를 줄이기 위해 가중치에 패널티를 부여하는 방법으로, 과대적합을 완화합니다. 대표적인 기법은 다음과 같습니다:

  • L1 정규화 (Lasso): 일부 가중치를 0으로 만들어 불필요한 특징을 제거하는 효과가 있습니다. (특징 선택)
  • L2 정규화 (Ridge): 모든 가중치를 0에 가깝게 작게 유지하여 모델의 복잡도를 제한합니다. (가중치 감소)

정규화를 통해 모델이 학습 데이터에 과도하게 적응하지 않도록 유도함으로써, 일반화 성능이 향상됩니다.

2주차: 딥러닝 심화

문제:

CNN(합성곱 신경망)에서 이미지의 특징(예: 윤곽선, 질감)을 추출하는 역할을 주로 담당하는 계층은 무엇인가요?

모범 답안:

합성곱 계층 (Convolution Layer)입니다. 교안의 'CNN 모델에 대한 개념' 파트에서 설명하듯, 합성곱 계층은 필터(커널)를 사용하여 이미지의 국소적인 특징을 추출하는 핵심적인 역할을 수행합니다.

문제:

딥러닝 모델 학습 시, 훈련 데이터에 대한 성능은 매우 높지만 새로운 테스트 데이터에 대한 성능은 낮게 나오는 현상을 무엇이라고 하며, 교안에서 소개된 해결 방법으로 가장 적절한 것은 무엇인가요?

모범 답안:

이 현상은 과적합(Overfitting)입니다. 교안의 '딥러닝 모델 학습의 문제점' 파트에서 설명하듯, 훈련 데이터에만 과도하게 최적화되는 현상은 과적합입니다. 이를 해결하기 위한 대표적인 규제 기법으로 드롭아웃(Dropout)을 적용할 수 있습니다.

문제:

교안에 따르면, 단층 퍼셉트론(Single-Layer Perceptron)이 해결할 수 없는 대표적인 문제인 'XOR 문제'의 근본적인 원인과, 다층 퍼셉트론(MLP)이 이를 해결하는 핵심 원리를 설명하시오.

모범 답안:

1. 단층 퍼셉트론의 한계 (원인):
단층 퍼셉트론은 본질적으로 **선형 분류기(Linear Classifier)**입니다. 즉, 데이터를 오직 하나의 직선(또는 고차원에서는 초평면)으로만 나눌 수 있습니다. XOR 문제는 데이터 분포상 어떤 직선을 그어도 두 클래스를 완벽하게 나눌 수 없는 **비선형(Non-linear) 문제**이기 때문에 단층 퍼셉트론으로는 해결할 수 없습니다.

2. 다층 퍼셉트론의 해결 원리:
다층 퍼셉트론(MLP)은 입력층과 출력층 사이에 **하나 이상의 은닉층(Hidden Layer)을 추가**하여 이 문제를 해결합니다. 이 은닉층은 입력 공간을 비선형적으로 변환하여, 원래 공간에서는 선형 분리가 불가능했던 데이터를 **새로운 특징 공간에서는 선형 분리가 가능한 형태**로 만들어주는 역할을 합니다. 결과적으로 여러 층을 쌓음으로써 복잡한 **비선형 결정 경계(Non-linear Decision Boundary)**를 학습할 수 있게 됩니다.
